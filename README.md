## 🚀 Featured Projects

### 🤖 Energy Consumption Prediction — ANN
> End-to-end regression model predicting Heating Load from building features using a custom Artificial Neural Network.

| Tech | Details |
|------|---------|
| Keras + TensorFlow | 8-input → 64/32/16 hidden layers (ReLU + Dropout) → Linear output |
| Adam + MSE | Trained on ENB2012 dataset · sub-2 MAE achieved |
| Streamlit | Deployed as interactive web app |

**Key wins:**
- 🔢 Full custom ANN architecture designed from scratch
- 📊 Complete pipeline: EDA → preprocessing → training → evaluation
- 🌐 Live Streamlit UI for real-time predictions

[![Repo](https://img.shields.io/badge/GitHub-energy--consumption--prediction-a78bfa?style=flat-square&logo=github)](https://github.com/subham23s/energy-consumption-prediction)

### 🔗 Blockchain-Based ML Model Integrity Verification
> Decentralized system to verify ML model files haven't been tampered with — combining blockchain, IPFS, and cryptographic hashing.

| Tech | Details |
|------|---------|
| Python + Flask | Backend API + web UI |
| SHA-256 | Cryptographic tamper detection on model files |
| IPFS (Pinata) | Decentralized model storage |
| Proof-of-Work | Custom mining implementation |

**Key wins:**
- ⛓️ Full PoW blockchain built from scratch in Python
- 🛡️ SHA-256 hash verification for ML model integrity
- 🌐 Flask web UI for uploading and verifying models
