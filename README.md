**⛓️ Blockchain ML Model Verifier**

Verifies ML model integrity using a custom blockchain — nobody can tamper with your model weights silently.

```
Upload Model File
    ↓
SHA-256 Hash Generated
    ↓
Proof-of-Work Mining
    ↓
Block Added to Chain
    ↓
IPFS Storage (Pinata)
```
🔐 Tamper-proof · 🌐 Flask UI
